Embed quotes in your wordpress sites either in widgets or inside your articles using this plugin. Elegant! Stylish and simply the best!

== Description ==
Highlights of Some useful features:

* Powered by cloud based quotes platform
* Easy to install packages with automatic updates.
* Great styles to choose from.
* You can pick from quote of the day categories or embed individual quotes.
* Can't find a quote you really love in our database? No problem! You can create them yourself in https://theysaidso.com and embed in all your sites. That is cloud based quotes platform for you!.
* New display styles added frequently.

== Installation ==

1.  Download the latest plugin version from the following URL
    https://theysaidso.com/extensions/wordpress/

2. Upload the plugin file to the `/wp-content/plugins/` directory. Alternatively you can search for "they said so" in wordpress plugins directory and click install.

3. Activate the plugin through the 'Plugins' menu in WordPress

4. Place tags like this 
     {theysaidso}display_style:'tso_pinched',qod_category: 'inspire' {/theysaidso}
   in your articles to insert quotes
5. You can also add a quote as a widget. Just drag the widget to the widget area and specify the quote category or the quote id in the widget options.

== Frequently Asked Questions ==

= I didn't find the quote that I was looking for in your site. How can I embed it?  =

Please login to the site here https://theysaidso.com/login and choose Create > Add a new quote from the top menu. When you are done you can get the quote id from the link displayed after the successful addition of the quote. Use that quote id ( Go to the quote page and click on the embed button below the quote to get the id) to embed it in your site.

= Is there a limit on how many quotes I can embed in an article? =

No. Go ahead and feel free to add how many ever you want.

= What resources are local and what resources are pulled from cloud? =

The plugin files are located locally. The style sheet is usually pulled from the cloud so that you always get the latest version. The quote itself lives in the cloud in They Said So platform and will be dynamically loaded on the client browser.
